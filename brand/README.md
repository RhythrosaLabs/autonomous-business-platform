# Brand Templates

This folder contains reusable brand assets for video production.

## Templates

### CTA End Card
- **Image**: `templates/cta_end_card.png` (1920x1080)
- **Video**: `templates/cta_end_card_3.5s.mp4` (3.5 seconds)

**Content**:
```
Now Available.
Visit our store today.
```

**Style**: 
- Black background
- White modern font
- Centered text
- Clean, professional aesthetic

## Usage

### Automatic Integration
The CTA end card is automatically appended to all videos generated by `static_commercial_producer.py`.

- 15-second commercial → becomes 18.5-second commercial with CTA
- The CTA plays after the main product content
- Provides clear call-to-action for viewers

### Manual Usage
```python
from brand.brand_template_generator import get_or_create_cta_video

# Get the CTA video (creates if doesn't exist)
cta_path = get_or_create_cta_video()

# Use in your video pipeline
from moviepy.editor import VideoFileClip, concatenate_videoclips

main_video = VideoFileClip("my_commercial.mp4")
cta_clip = VideoFileClip(cta_path)
final = concatenate_videoclips([main_video, cta_clip])
final.write_videofile("commercial_with_cta.mp4")
```

### Regenerating Templates
```bash
cd brand
python3 brand_template_generator.py
```

## Customization

To customize the CTA text or styling, edit `brand_template_generator.py`:

```python
# Change text
create_cta_end_card(
    line1="Limited Time Only.",
    line2="Shop now at our store.",
    font_size=80,
    bg_color=(0, 0, 0),  # RGB
    text_color=(255, 255, 255)  # RGB
)

# Change duration
create_cta_video(duration=5.0)  # 5 seconds instead of 3.5
```
